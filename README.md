## Data Science Work

During the 2021 fellowship, completed under the guidance of mentors [Prasun Kumar Mishra](https://www.kaggle.com/prasunmishra) and [Julia Elliott](https://www.kaggle.com/juliaelliott):

**Coursework**
- Completed the Kaggle Learn course on Data Visualization

**Dataset Research**
- Identified and catalogued approximately 100 datasets relevant to mutual aid and artist resources
- Sources: CSV files, HTML scrapes, API/RSS feeds from Kaggle, data.gov, arts.gov, Google Dataset Search, Twitter API, and community mutual aid aggregators

**Architecture & Pipeline Design**
- Designed a proposed data pipeline using Python and BeautifulSoup to process raw HTML from resource sites and generate structured DataFrames
- Planned integration with Jekyll via GitHub Pages for public-facing output
- Explored a conversational chatbot UI layer for user-facing resource queries

**Database & Feature Planning**
- Prototyped "To-Be" database table designs outlining the mutual aid directory's data structure
- Defined feature requirements: resource listings, artist directory, donor/sponsor functionality, geographic filtering exportable by metro area
- Documented privacy considerations for mutual aid organizers (referencing EFF guidelines)

**Planning Worksheet**
- Maintained a full project planning worksheet tracking milestones, datasets, research links, blockers, and learning plan — preserved in the [Google Drive project folder](https://drive.google.com/drive/u/1/folders/1QodJ016_nw7tzJJ9T8KN6AYhJN0JRI_P)

## My Beautiful Device

*A cyberpunk novel of analog soul, mutual aid, and human spirit.*

An antidystopian sci-fi fantasy about a hero trying to obtain a government-issued emotional support robot — and what happens when the system keeps saying no. The themes mirror the mutual aid work directly: a story about what it means to need help, to ask for it, and to keep going when the system isn't built for you.

Written and recorded as an audiobook in 2019–2020, before the modern generative AI era. The YouTube timestamps are proof of the work's human, analog origins. Published March 4, 2026.
